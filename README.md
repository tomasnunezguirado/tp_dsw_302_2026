Título del proyecto:
Sistema de Gestión para Pequeños Negocios

Descripción general:
El proyecto consiste en el desarrollo de una aplicación web destinada a la gestión de pequeños negocios. El sistema permitirá organizar y centralizar la información relacionada con productos, clientes y ventas, facilitando el control y la administración diaria.

Funcionalidades principales:

Gestión de productos (alta, baja, modificación y consulta).

Control de stock disponible.

Registro y gestión de clientes.

Registro de ventas.

Consulta de historial de ventas.

Visualización de alertas de stock bajo.
